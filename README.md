# 📝 Nepali OCR Web App

A simple  web application to extract Nepali text from images using **Tesseract OCR**, **OpenCV** and **Streamlit**.

🔗 **Live Demo:** https://nepali-ocr.streamlit.app/

---

## 🚀 Features

- 📤 Upload JPG, PNG, JPEG images  
- 🔍 Extract Nepali text using OCR (`lang='nep'`)  
- 🖼️ View original and processed image  
- 📄 Display extracted text  
- ⬇️ Download extracted text as `.txt`  
---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- OpenCV  
- Tesseract OCR  
- NumPy  
- Pillow
